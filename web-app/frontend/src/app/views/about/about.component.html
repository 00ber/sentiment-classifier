About the project

This application is designed to discern the sentiment of textual content. It's a straightforward tool that categorizes any given text as either positive or negative.

For this project, the cardiffnlp/twitter-roberta-base-sentiment-latest model was fine-tuned on the IMDB Dataset of 50K Movie Reviews dataset from Kaggle.
